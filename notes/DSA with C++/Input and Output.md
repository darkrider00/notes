## skeleton

```cpp
#include<iostream>

int main() {
    // Your code here
    return 0;
}
```

## output
- we use cout fuction from iostream lib 
- std:cout << "hey, santhosh"
```cpp
#include<iostream>

int main() {
    std::cout << "Hey, santhosh";
    return 0;
}

```

```cpp
    std::cout << "Hey, 
Striver!";

Striver!";

}
```
