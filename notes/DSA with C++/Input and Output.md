`#include<iostream>` is used for input and output 
`#include <math.h>`  is used for math functions

## skeleton

```
#include<iostream>

int main() {
    // Your code here
    return 0;
}
```

## output
- we use cout fuction from iostream lib 
- std:cout << "hey, santhosh"
```cpp
#include<iostream>

int main() {
    std::cout << "Hey, santhosh";
    return 0;
}

```
